About the website

This website is a showcase of my skills as a frontend developer. It is built with Angular and the Star Wars API. It is a single page application with a responsive design. It makes use of the ngrx library to open a dialog with the details of the selected item. For the styling I used the Angular Material library. As directed each page is divided into modules and lazy loaded. Get requests to update the data on the cards are cached on the page so that they are sent only once. All requests are made via the CoreDataService to hide the low level details of the API. The CoreDataService is a singleton service that is injected into the components that need it. The updated data in the details dialog is turned into a link that can be clicked to open the details of that selected item. The details dialog is a reusable component that is injected with the data of the selected item.
